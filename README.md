# Expense Tracker API

A simple REST API built using Flask to manage daily expenses.
---
## Features
✔ Add Expense  
✔ View Expenses  
✔ Delete Expense  
✔ Show Total  
✔ Filter by Category  
---
## Tech Stack

- Python
- Flask
- SQLite
- REST API
---
## Project Structure
expense-tracker-api/
│── app.py

│── requirements.txt
│── README.md
---
## How to Run
1. Install dependencies
pip install -r requirements.txt
2. Run the app
python app.py
