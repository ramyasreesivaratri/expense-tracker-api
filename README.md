# Expense Tracker API

A simple REST API built using Flask to manage daily expenses.
---
## Features
1. Add Expense  
2. View Expenses  
3. Delete Expense  
4. Show Total  
5. Filter by Category  
---
## Tech Stack
- Python
- Flask
- SQLite
- REST API
---
## Project Structure
expense-tracker-api/
│── app.py
│── expenses.db
│── requirements.txt
│── README.md
---
## How to Run
1. Install dependencies
pip install -r requirements.txt
2. Run the app
python app.py
