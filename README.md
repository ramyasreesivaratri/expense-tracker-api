# Expense Tracker API
A simple REST API built using Flask to track daily expenses.
---
## Features
- Add Expense
- View All Expenses
- Update Expense
- Delete Expense
- SQLite Database Integration
---
## Tech Stack
- Python
- Flask
- SQLite
- REST API
